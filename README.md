# Бот репетитора английского

Telegram-бот на Python (aiogram 3): запись на уроки, напоминания, оплата абонементов, домашние задания. С админ-панелью для учителя.

## Что умеет

**Для учеников:**
- 📅 Записаться на свободный слот
- 🔔 Смотреть свои записи, переносить и отменять
- 💳 Видеть баланс оплаченных уроков и реквизиты для оплаты
- 📚 Получать ДЗ и сдавать ответы файлом/текстом
- Напоминания за 24 часа и за 2 часа до урока
- Напоминание об оплате, когда уроки заканчиваются

**Для учителя (админ-панель, команда `/admin`):**
- ➕ Добавлять слоты расписания
- 🗓 Видеть все записи
- ✅ Отмечать проведённые уроки (списывает 1 урок с баланса ученика)
- 💰 Отмечать оплату (начислять уроки на баланс ученика)
- 🔔 Напоминать конкретному ученику об оплате (с реквизитами)
- 📝 Выдавать домашние задания
- 👥 Список учеников с балансом уроков
- 📣 Рассылку всем ученикам
- ☀️ Утреннюю сводку уроков на день

## Установка (один раз)

1. **Создай бота.** Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → получи **токен**.
2. **Узнай свой ID.** Напиши [@userinfobot](https://t.me/userinfobot) → он пришлёт твой числовой **ID**.
3. **Установи Python 3.10+** (если ещё нет): https://python.org
4. **Установи зависимости.** В терминале в папке проекта:
   ```
   pip install -r requirements.txt
   ```
5. **Настрой `.env`.** Скопируй `.env.example` в файл `.env` и впиши `BOT_TOKEN` и `ADMIN_ID`.

## Запуск

```
python main.py
```

Бот работает, пока открыт терминал. Останов — `Ctrl+C`.

## Как работает оплата

Онлайн-приёма денег нет — реквизиты у учеников уже есть. Бот только показывает реквизиты и напоминает об оплате:

1. Ученик оплачивает по твоим реквизитам (вне бота).
2. Ты жмёшь «💰 Отметить оплату» → выбираешь ученика → вводишь число уроков. Они зачисляются на баланс.
3. После каждого урока («✅ Провести урок») с баланса списывается 1 занятие.
4. Когда остаётся 1 или 0 уроков, бот сам напоминает ученику об оплате с реквизитами.
5. Можно напомнить вручную — «🔔 Напомнить об оплате».

Реквизиты задаются в `.env` в переменной `PAYMENT_DETAILS`. Перенос строки — через `\n`.

## Файлы проекта

| `handlers_admin.py` | админ-панель |

## Как обычно проходит работа

1. Ты добавляешь слоты на неделю (➕ Добавить слоты).
2. Ученик записывается → тебе приходит уведомление.
3. Бот сам напоминает ученику за сутки и за 2 часа.
4. После урока жмёшь «✅ Провести урок» — списывается 1 занятие.
5. Выдаёшь ДЗ (📝), ученик сдаёт ответ — он приходит тебе.
6. Когда уроки заканчиваются, бот предлагает продлить абонемент.
